## Project Amethyst Engine 

Our project is to devlop a basic graphics engine using the glut toolkit for rendering opengl.

## Goals
Create rendering for obj models
allow for animation of models
camera controls
menu controls

## Dependencies
Codeblocks with MinGW.
Git for windows 
freeglut

## Installation
Instructions for Installing:
Download and install Codeblocks, make sure if you are using Codeblocks to download the version with mingw, which is labeled codeblocks-xx.xxmingw-setup.exe. Once downloaded go ahead and just run the installer.
Download freeglut, make sure it is the MinGW version. Once downloaded instructions for installing with codeblocks can be found here; wiki.codeblocks.org/index.php/Using_FreeGlut_with_Code::Blocks
Download Git for windows, use the wizard to install. You will most likely want to use all the default options. Once this is done you can use the git terminal to use git commands.

Links for downloading:
Codeblocks: www.codeblocks.org/downloads/26
Freeglut: www.transmissionzero.co.uk/software/freeglut-devel/
Git for windows: https://git-scm.com/download/win
