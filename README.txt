Dependencies:
Codeblocks_mingw version
Git for windows 
freeglut

Instructions for Installing:
Download and install Codeblocks, make sure if you are using this to download the version with mingw, which is labeled codeblocks-xx.xxmingw-setup.exe
Download freeglut, make sure it is the MinGW version. Instructions for installing with codeblocks can be found here; wiki.codeblocks.org/index.php/Using_FreeGlut_with_Code::Blocks
Download Git for windows, go through the wizard to install you will most likely want to use all the default options. Once this is done you can use the git terminal to use git commands

Links for downloading:
Codeblocks: www.codeblocks.org/downloads/26
Freeglut: www.transmissionzero.co.uk/software/freeglut-devel/
Git for windows: https://git-scm.com/download/win
